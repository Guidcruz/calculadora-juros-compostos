# Calculadora de Juros Compostos

Aplicação web para simular o crescimento de investimentos com juros compostos, aportes mensais e visualização da evolução ao longo do tempo.

## Funcionalidades

- **Capital inicial e aporte mensal** — comece do zero ou com qualquer valor inicial
- **Taxa mensal ou anual** — escolha a periodicidade da taxa de juros
- **Período em meses ou anos** — flexibilidade total no prazo
- **Máscara de moeda** — os inputs formatam automaticamente o valor ao sair do campo
- **Gráfico de evolução** — linha do montante vs. total investido com crosshair ao passar o mouse
- **Tabela detalhada** — acompanhe saldo, total investido e juros acumulados por período
- **Modo escuro** — alternável pelo botão no header
- **Suporte a PT/EN** — troca o idioma e a moeda (R$ / $) com um clique
- **Responsivo** — adaptado para desktop, tablet e mobile

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Preencha os campos de capital, aporte, taxa e período
3. Clique em **Calcular**
4. Analise o resultado no gráfico e na tabela

## Tecnologias

- HTML, CSS e JavaScript puro
- [Chart.js](https://www.chartjs.org/) para o gráfico
